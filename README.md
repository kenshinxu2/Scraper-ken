# Telegram Video Cover Bot

Changes video covers instantly (0.04 seconds) while preserving captions.

## Setup on Railway

1. Fork this repo to your GitHub
2. Create new project on Railway
3. Deploy from GitHub repo
4. Add environment variable: `BOT_TOKEN`
5. Deploy!

## Environment Variables

- `BOT_TOKEN` - Your Telegram Bot Token from @BotFather

## Usage

1. Send video to bot
2. Send HD cover image
3. Get video back with new cover instantly!

## Features

- ✅ 0.04 second cover change
- ✅ HD cover support
- ✅ Caption preserved
- ✅ Same format output (1,2,3,4)
- ✅ Batch processing support
